2D Side-Scroller Project

The Game / Side Quest

Without giving too much away, I'm currently developing a 2D Side-Scroller game myself. The game will be a mix of elements from some of my favorite 2D side-scroller games in a cyberpunk setting. It will contain parts of Mega Man X, Super Metroid Prime and Hollow Knight all in a futuristic setting.

Currently I have: concept art, storyline and most of the C# scripts setup, but the game is still a WIP. At this moment I'm mostly looking for help with sprite sheets and level design. Would be great to have a team of artists and level designers to help me get the game to the next level.

Game Development Lifecycle
